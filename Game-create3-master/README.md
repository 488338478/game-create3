# Game-create3

2D side-scrolling narrative game prototype (picture-book / fairy-tale tone) built with Unity `2022.3.62f3c1`.

## 唯一维护入口
- 项目手册：`Docs/Project_Handbook.md`
- 文档规则：`Rule DOC-001`（任何功能/配置/流程变化都必须同步更新手册）

## 快速开始
1. 使用 Unity Hub 以 `2022.3.62f3c1` 打开本项目。
2. 打开 `Assets/Scenes/SampleScene.unity`。
3. 按 `Docs/Project_Handbook.md` 的装配步骤完成基础配置。
